Simple Netflix clone website.

- React
- CSS
- Firebase

## Setup

To run this project, install it locally using npm:

1. Clone the repo

```bash
git clone https://github.com/Globalkmaria/netflix_clone.git
```

2. Go into the folder

```bash
cd netflix_clone
```

3. Install NPM packages

```bash
npm i
```

4. Start server

```bash
npm start
```
